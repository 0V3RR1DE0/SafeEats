SafeEats

A food restriction scanner app

Description

SafeEats is a mobile application designed to help users identify and manage food restrictions by scanning barcodes on food products. The app provides detailed information about the product, including its name, brands, ingredients, and potential allergens. Users can also manually enter EAN codes to fetch product details.

Features

  • Barcode Scanning
    • Automated Scanning: The app uses the device's camera to scan barcodes on food products.
    • Manual Entry: Users can manually enter EAN codes if the barcode cannot be scanned.
  • Product Information
    • Product Name: Displays the name of the food product.
    • Brands: Shows the brand(s) of the food product.
    • Ingredients: Lists the ingredients of the food product.
    • Allergens: Highlights any potential allergens present in the food product.
  • User Settings
    • Food Restrictions: Users can customize their food restrictions by adding, editing, or deleting specific ingredients or allergens.

Installation

Follow these steps to install and use the SafeEats APK file on your Android device:

1. Prepare Your Device

Before installing the APK file, you need to allow installations from unknown sources:

  1. Open the Settings app on your Android device.
  2. Navigate to Security or Privacy (the exact name may vary).
  3. Find and enable the option Install unknown apps or Unknown sources.
  4. Select the app (like your file manager or browser) that you will use to install the APK and toggle the switch to allow installations from that source.

2. Download the APK File

Download the APK file from the following link:

  • Download SafeEats APK

3. Install the APK File

Once the APK file is downloaded, follow these steps to install it:

  1. Open the File Manager app on your device.
  2. Navigate to the Downloads folder or the location where the APK file is saved.
  3. Tap on the APK file. You might see a prompt asking if you want to install the app. Tap Install.
  4. Wait for the installation process to complete. Once done, you can tap Open to launch the app or find it in your app drawer.

4. Using the App

After installation, you can start using the app:

  1. Open the app drawer or home screen on your device.
  2. Find the SafeEats app icon and tap on it to launch the app.
  3. Follow any initial setup instructions provided by the app, such as signing in or setting preferences.

5. Troubleshooting

If you encounter issues:

  • Installation Issues: Ensure unknown sources are enabled and that the APK is compatible with your device.
  • App Crashes or Errors: Restart your device, check storage and memory, or update the APK if available.
  • Security Warnings: Download APKs only from trusted sources.
